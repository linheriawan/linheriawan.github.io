❓ Frequently Asked Questions

Common questions about Aksara Notes

Q: I forgot my master password. What do I do?
A: Unfortunately, master passwords cannot be recovered. You'll need to reset the app, which will erase all data. This is by design for security.
Q: Can I sync data across devices?
A: Currently, data is stored locally for security. Cloud sync is planned for future updates with end-to-end encryption.
Q: How do I backup my data?
A: The backup feature is coming soon! You'll be able to create encrypted backups to your device storage.
Q: Why do I need to authenticate so often?
A: This is for security. You can adjust auto-lock settings in Security Settings to reduce frequency.
Q: Can I import data from other apps?
A: The database builder allows creating tables for any data structure. Manual import tools are planned for future updates.
Q: Is my biometric data stored in the app?
A: No! Biometric authentication uses your device's secure enclave. No biometric data is stored in Aksara Notes.
Q: How do I create complex database relationships?
A: Currently, tables are independent. Relationship features are planned for future updates.
